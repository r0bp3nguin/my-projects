# Application Cross Training pour Garmin

## Description

Application de cross training développée pour les montres Garmin, permettant de suivre et d'enregistrer les séances d'entraînement crossfit directement depuis une montre connectée.

## Lien de l'application

L'application est disponible sur le Garmin Connect IQ Store :

[Cross Training App - Garmin Connect IQ](https://apps.garmin.com/apps/2ea0229d-f90a-4c81-bad0-eec8114200f7)
